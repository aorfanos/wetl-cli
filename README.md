# wetl-cli
A simple, one-function CLI for uploading files to WeTransfer.

Uses: WeTransfer API (https://developers.wetransfer.com/documentation)
Requirements: curl,jq

### Usage
```
wetl-cli <file>

#Example:
wetl-cli ~/Downloads/file.zip
```
